# 🍔 Texi Burger Restaurant

A simple and responsive **restaurant website** built using **raw HTML, CSS, and JavaScript**.
The website showcases a burger restaurant with menu items, restaurant information, and a clean modern UI.

This project demonstrates **front-end web development fundamentals without frameworks**.

---

## 🚀 Features

* 🍔 Restaurant landing page
* 📜 Menu section with burger items
* ℹ️ About section for restaurant information
* 📞 Contact section
* 📱 Responsive layout
* ⚡ Interactive elements using JavaScript
* 🎨 Clean and simple UI

---

## 🛠️ Built With

* **HTML5** – Page structure
* **CSS3** – Styling and layout
* **JavaScript (Vanilla JS)** – Interactivity

## ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/yourusername/texi-burger-restaurant.git
```

2. Navigate to the project folder

```bash
cd texi-burger-restaurant
```

3. Open `index.html` in your browser.

No additional dependencies or build tools are required.
